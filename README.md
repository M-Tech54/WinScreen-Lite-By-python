# 🎥 WinScreen-Lite 



يا هلا بيك يا زميلي! **WinScreen-Lite** هو برنامج بسيط وخفيف جداً معمول عشان يسجل شاشة الكمبيوتر (الويندوز) بلمسة واحدة، ومن غير ما يهنّج الجهاز أو يستهلك الرام زي البرامج الكبيرة المعقدة.

### ✨ المميزات:
* **خفيف طلقة:** مش بيتقل الجهاز وقت اللعب أو الأكواد.
* **بسيط على المفتاح:** واجهة سهلة فيها زرار ابدأ (Start) وزرار وقف (Stop).
* **حفظ تلقائي:** بيسجل الفيديو بجودة عالية ويحفظه فوراً بصيغة `MP4`.
* **مفتوح المصدر (Open Source):** تقدر تعدل عليه وتطور فيه براحتك!

### 🛠️ المكتبات المستخدمة (Dependencies):
البرنامج ده مبني بلغة بايثون `Python` وبيعتمد على المكتبات دي:
* `opencv-python` (cv2) - عشان يجمع الصور ويطلعها في فيديو.
* `pyautogui`  عشان يلقط الشاشة بسرعة عالية.
* `tkinter` - عشان الواجهة والشاشات الرايقة.

### 🚀 ازاي تشغله عندك:
1. اتأكد إنك منزل بايثون على جهازك.
2. افتح الـ Terminal ونزل المكتبات دي:
```bash
   pip install opencv-python pyautogui
